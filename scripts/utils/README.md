# 🧪 Utilities

Helper scripts for daily operations.

## Contents

- `generate-structure.sh` → Scaffolds the platform-engineering-kit base folder structure and starter READMEs.
- `lib/ci/run-guardian-audit.sh` -> Runs repository-wide CI governance checks and fails on policy violations.
- `lib/ci/validate-english-content.sh` -> Fails when Portuguese/non-English content is detected in language-governed repository areas.
- `lib/ci/validate-script-naming.sh` -> Fails when shell scripts under `scripts/` are not in kebab-case.
- `lib/` -> Reusable shell libraries for logging, package management and common script functions.

## Purpose

Simplify common tasks.
