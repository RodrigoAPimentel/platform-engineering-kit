# 🧪 Utilities

Helper scripts for daily operations.

## Contents

- `generate-structure.sh` → Scaffolds the platform-engineering-kit base folder structure and starter READMEs.
- `validate-english-content.sh` -> Fails when Portuguese/non-English content is detected in language-governed repository areas.
- `lib/` -> Reusable shell libraries for logging, package management and common script functions.

## Purpose

Simplify common tasks.
